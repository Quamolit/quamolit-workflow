
Quamolit Workflow
----

> project template for [quamolit.calcit](https://github.com/Quamolit/quamolit.calcit).

### Usages

Install Node.js and [Calcit](https://github.com/calcit-lang/calcit_runner.rs) first. And load dependencies in `~/.config/calcit/modules/`. The compile and run:

```bash
yarn
cr --emit-js -1
yarn vite
```

Also check out `.github/workflows/upload.yaml` for an example.
